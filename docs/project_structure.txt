## Описание компонентов

### Бэкенд (FastAPI)

| Файл | Назначение |
|------|------------|
| `main.py` | Инициализация FastAPI, CORS, роутеры |
| `db.py` | Подключение к ChromaDB (векторная база) |
| `schemas.py` | Модели запросов/ответов API |
| `rag.py` | Endpoint `/api/query` для поиска ответов |
| `documents.py` | Endpoints для загрузки документов |
| `rag_service.py` | RAG логика: эмбеддинги, поиск, генерация |
| `load_document.py` | Скрипт загрузки документов в базу |

### Фронтенд (Vue 3)

| Файл | Назначение |
|------|------------|
| `App.vue` | Корневой компонент |
| `main.js` | Точка входа Vue приложения |
| `QueryForm.vue` | Компонент отправки запросов |
| `api.js` | HTTP клиент для взаимодействия с бэкендом |

## API Endpoints

### Backend (port 8000)

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/query` | Поиск ответа по документам |
| GET | `/health` | Health check |

## Технологии

- **Бэкенд**: FastAPI, LangChain, ChromaDB
- **Фронтенд**: Vue 3, Vite, Tailwind CSS
- **Эмбеддинги**: intfloat/multilingual-e5-large (routerai.ru)
- **LLM**: google/gemini-3.1-flash-lite-preview (routerai.ru)
- **OpenAI Endpoint**: https://routerai.ru/api/v1
